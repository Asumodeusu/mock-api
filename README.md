# mock-api

To install dependencies:

```bash
bun install
```

Чтобы запустить нужно - открыть консоль и ввести:

```bash
bun run start:api
```

Так же ввести порты для проверки данных в браузере:

```bash
1 - http://localhost:3000
2 - http://localhost:3000/api/dictionaries/activity-types 
3 - http://localhost:3000/api/employees
4 - http://localhost:3000/api/schedule/all
```


This project was created using `bun init` in bun v1.1.17. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
